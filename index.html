学习爬虫之前，先看看大神写的技能表，虽然后面方向不是我们要去走的，但是前面很大一部分是需要我们去拥有的。

研发技能表

决定了学习爬虫，那就找个标准（月薪20K-45K）去学习吧。

Python爬虫路线-转自头条

  1. python部分
  2. Linux部分
  3. Web部分
  4. 网络部分
  5. 爬虫部分

Python部分

1.python基础
2.python进阶

1. python基础

廖雪峰Python学习指南

1.1 基本语法（基本知识，数据类型，流程控制）
1.2 函数（内置函数，自定义函数，递归，闭包，lambda匿名函数）
1.3 模块（常用模块，内置模块）
1.4 I/O 基础（文件操作，StingIO， BytesIO）

2. python进阶

2.1 面向对象（基础，魔术方法，描述器，数据类型，上下文管理器，迭代器）
2.2 装饰器，惰性器
2.3 高级模块
2.4 并发编程（多线程和多进程）
2.5 并发编程（协程，异步IO）
2.6 其他总结

Linux部分

这一部分目标暂时知道怎么会用Linux，等爬虫其他知识掌握后，再深入学习一下。

1.1 获取与安装
1.2 基本操作（目录操作，目录结构，文件操作，工具命令，用户及文件权限管理）
1.3 进程及任务管理等
1.4 vim编辑器
1.5 包管理

Web部分

我们现阶段目标是要知道前端是怎么工作的，知道其原理，能看懂前端代码，还可以去做点点修改最好。

  1. HTML
  2. CSS
  3. JavaScript 基本语法
  4. Ajax
  5. 浏览器原理

网络部分

这一块我们不从网络的角度去看待，我们从软件研发的角度去学习

  1. HTTP、HTTPS
  2. TCP/IP
  3. 抓包分析数据

爬虫部分

有了前面的基础，这一块才是我们需要花费大量精力的地方

1.爬虫基础
2.爬虫进阶

1. 爬虫基础

1.1 浏览器&请求与相应（Request&Response GET&POST，cookie&session）
1.2 XPath & css选择器 & 正则
1.3 爬虫和反爬虫
1.4 开发工具（Fiddler, pycharm, Chrome 开发者工具，Charles， Appium等）
1.4 第三方库（Requests等）

2. 爬虫进阶

2.1 爬虫框架 scrapy
2.2 代理池实现
2.3 模拟登陆
2.4 APP数据抓取（Charles，Mitmproxy,MitmDump,Appium等）
2.5 分布式
